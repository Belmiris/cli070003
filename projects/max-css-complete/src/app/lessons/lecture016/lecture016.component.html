Using a selector in a .css file

Choose your plan

  • We can set styles by selecting html elements.
  • We can set styles by selecting classes assigned to html elements. These are defined with a dot (.) in front of the class name.
  • You can set a universal style that applies to all elements. These are represented by an asterisk (*).
  • We can set styles by selecting the ids given to html elements. These are defined with a hash (#) in front of the id name.
  • We can set styles by selecting the attributes given to html elements. These are by enclosing the attribute name in brackets [attr-name].
