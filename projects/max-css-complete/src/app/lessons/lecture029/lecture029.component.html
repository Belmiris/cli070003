Shorthand Properties

Combine the values of multiple properties into a single property

Seperate properties

border-width: 2px;
border-style: dashed | solid;
border-color: orange;

If you omit a property, the default property for that element will be used.

Shorthand Property

border: 2px dashed orange;

Margin is an example of a property with many shorthand properties

margin: 5px 10px 5px 10px; /* top right bottom left */
margin: 5px; /* top & bottom right & left */
margin: 5px; /* same value for all sides */
