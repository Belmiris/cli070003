Applying Additional Styles & Importing Google Fonts

We use sans-serif here which uses the default font specified in your browser.

See your browser'ssettings to find what the default fonts are.

In Chrome there are settings for Standard, Serif, Sans-Serif and Fixed-Width Font Families.

You cannot depend on a browser to have a specific font. If you need to do theis you can import fonts from Google. https://fonts.google.com

Here we will import the font 'Anton'.

Click to plus next to the font you want on the page and a popup will appear with a link. https://fonts.googleapis.com/css?family=Anton
