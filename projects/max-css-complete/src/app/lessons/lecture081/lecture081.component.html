Positioning Elements

  • The position property is handled automatically to ensure document flow. The default is position: static;
  • It is possible to change the position property to allow you to place elements differently
  • Other possible values are below:
  • position:absolute;
  • position:relative;
  • position:fixed;
  • position:sticky;
  • If we change the position property, we need to be aware of the positioning context
  • What will the values we supply in the top, left, width, height use as a reference point?
