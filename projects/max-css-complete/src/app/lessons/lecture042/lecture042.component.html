  • There are Pseudo Classes and there are Pseudo Elements
  • Psuedo classes allow us to define a style for the 'state' of an element (i.e. hover, active)
    class: name
    A full list of Psuedo classes can be found on the mdn website
  • Psuedo elements allow us to define a style for a specific part of an element
    : : element name
    A full list of Psuedo elements can be found on the mdn website
  • NOTE: The ::before and ::after psuedo elements allow you to actually add and style additional content!
