There are four types of combinators

  • Adjacent Sibling: h2 + p { }; Applies to all paragraphs directly following an h2 tag.
  • General Sibling: h2 ~ p { }; Applies to all paragraphs following an h2 tag (need not be adjacent).
  • Child: div > p { }; Applies to all paragraphs that are immediate children of a div.
  • Descendant: div p { }; Applies to all children of a div.
