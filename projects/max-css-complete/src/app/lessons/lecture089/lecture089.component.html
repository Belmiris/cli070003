Working with Overflow and Relative Positioning

A way to handle child elements with postion: relative; is to hide the parts that pass the parents boundaries

Set the parent element's property to overflow: hidden; to hide children that pass the boundaries.

WARNING

If you set the position to relative of a top level element and then try to set the body element property overflow: hidden; it will have no effect!

This is due to a default behavior of browsers when the overflow property on the body will be transfered to the html element.

By definition, nothing can 'overflow' the html boundary.

WARNING 2

Stupidly, if you set the overflow property of the html element AND the body element to hidden it will work as 'expected'.
