  • box-sizing is a property that tells the browser how to apply width and height properties
  • The default value for box-sizing is content-box
  • content-box means that the width and height are applied to only the content and the padding, border & margin are added seperately
  • border-box means we want to apply the sizes to the border and size the content to fit.
  • Despite content-box being the default property, usually you will want to set the box-sizing to border-box because it makes more sense.
  • Browsers will override the box-sizing property if you try to override the property of the body to make all elements use box-sizing.
  • In order to make all elements use box sizing you will have to use the universal selector (*)
